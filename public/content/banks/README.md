Please download the following bank logos and save them in this folder:

1. **DBBL (Dutch Bangla Bank)** - Save as `dbbl.png`
   - Visit: https://www.dutchbanglabank.com/
   
2. **City Bank** - Save as `city-bank.png`
   - Visit: https://www.thecitybank.com/
   
3. **BRAC Bank** - Save as `brac-bank.svg` or `brac-bank.png`
   - Visit: https://www.bracbank.com/
   
4. **EBL (Eastern Bank Limited)** - Save as `ebl.png`
   - Visit: https://www.ebl.com.bd/
   
5. **Islami Bank** - Save as `islami-bank.png`
   - Visit: https://www.islamibankbd.com/
   
6. **Prime Bank** - Save as `prime-bank.png`
   - Visit: https://www.primebank.com.bd/

## Instructions:
1. Visit each bank's website
2. Right-click on their logo
3. Save the image to this folder with the filename mentioned above
4. Preferred format: PNG with transparent background
5. Recommended size: Height around 40-60px

The landing page is already configured to use these local paths.
